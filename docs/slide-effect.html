Slide effect


The slide toggle effect is an animated to radio button effect mostly used for auto scrolling slides. In this situation a each checkbox are automatically switched based on specified intervals. This is achieved by setting the data-bind attribute to slide. An example is shown below:

Slide Effect HTML CSS 
  <!--parent checkbox (group)-->
  <div data-role="checkbox-list" data-bind="slide">
  
    <!--first checkbox section-->
    <div data-role="checkbox">
      <div checkbox data-anime="fadeIn" data-assign=":marker2" data-color="@#615c988d|lime">
      
        <div marker></div>
        <div marker></div>
        
      </div>
      <input type="checkbox" value="C1">
    </div> 
    
    <!--second checkbox section-->
    <div class="rotate" data-role="checkbox">
      <div checkbox data-anime="rubberBand infinite" data-assign=":marker2" data-color="@#615c988d|gold">
      
        <div marker></div>
        <div marker></div>
        
      </div>
      <input type="checkbox" value="C2">
    </div>
    
    <!--third checkbox section-->
    <div data-role="checkbox">
      <div checkbox data-anime="bounceIn|flash infinite" data-assign=":markers" data-color="@#615c988d|lime">
      
        <div marker></div>
        <div marker></div>
        
      </div>
      <input type="checkbox" value="C3">
    </div>
    
  </div>
  
[data-role="checkbox-list"] {
    display: inline-flex;
    gap: .5em;  
  }
    
  [data-role="checkbox-list"] [marker] {
    width: 20px;
    height: 20px;  
  }
    
  [checkbox] {
    display: inline-flex;
    border: dotted 2px;  
    border-radius: 100vh;  
    padding: 4px;  
  }
    
  [checkbox] [marker] {
    width: 20px;
    height: 20px; 
    border-radius: inherit;    
    background-color: currentColor;    
  }
  
  .rotate[checkbox][checked="checked"] {
      animation: rotate 5s infinite linear;;
  }
  
  @keyframes rotate {
  
      0% {
        transform: rotate(0deg);
      }
      
      100% {
        transform: rotate(360deg);
      }
      
  }
From the sample above, each checkbox is toggled based on specific interval. The default interval is 2500 milliseconds eqivalent to 2.5 seconds. This interval can be modified from the data-bind attribute by setting milliseconds after the "slide" option. An example is shown below:

From the sample above, the slide animation becomes faster because the interval was reduced to 500 milliseconds. Since each checkbox is capable of running it's own callback function, we can easily control the event which each checkbox runs when it is toggled on. However, the checkbox plugin makes this easier by providing a "listId" method that helps in determining the checkbox that is currently active. A better explanation of this is revealed in the sample below:
Scroll Effect HTML CSS JS 
  <!--parent checkbox (group)-->
  <div data-role="checkbox-list" data-bind="slide-500">
  
    <!--first checkbox section-->
    <div data-role="checkbox">

<script>
let checkbox = new CheckBox(false);
  
checkbox.check({

  assign: {
    alter: (dataAnime) => {
    
        dataAnime.unshift('animated');
        return dataAnime.map( el => 'animate__' + el );
        
    }
  }

});

/*callback function*/
function scroll(e){

  if (e.checked) {

    /*if a radio checkbox is checked*/
    
    let checkboxId = e.listId; /*get the checked checkbox id*/ 
    
    let slides = document.querySelectorAll(".slides .slide");
    
    slides.forEach(slide => {
      
      slide.classList.remove("active")
      
    });
    
    /* add active class to the only checked checkbox */
    slides[checkboxId - 1].classList.add("active");

  }

}
</script>

From sample above, we did not need to add an event on the slides. We only took advantage of the checkbox callback functions to animate the slides and we can easily tune up or tune down the animation interval easily without adding an extra javascript code. This shows how resourceful the checkbox plugin can be.
