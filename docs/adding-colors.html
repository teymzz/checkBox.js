Adding Colors

Colors are defined through the data-color attribute and they are primarily applied to labels. A sample structure of this is shown below

single color HTML CSS 
<div checkbox data-color="red" data-label="Checked|Unchecked" data-size="20px">
  <div marker=""></div>
</div>
<input type="checkbox">
<div data-access="label"></div>

.checkbox-wrapper {
  display: inline-flex;
  gap: .5em   
}

In the sample structure above, the label field is assigned a color of “red” which is specified through the data-color attribute. This means that the label texts will inherit the red color. The data-color also supports a maximum of two alternating colors. A sample format of this is shown below:

double colors HTML CSS 
<div class="checkbox-wrapper">
  <div checkbox data-size="20px" data-color="red|lime" data-label="Unchecked|Checked">
    <div marker="" style="background-color:currentColor"></div>
    <div marker="" style="background-color:currentColor"></div>
  </div>
  </div>
  <input type="checkbox">
  <div data-access="label"></div>
</div>

.checkbox-wrapper {
  display: inline-flex;
  gap: .5em   
}

In the double colors sample structure above, two colors were defined through the data-color attribute. From the html response, we can see that only the label text color is toggled from red to lime and this does not affect the marker's white background color. The red color is only assigned to the label field when the checkbox is unchecked while the lime color is applied when the checkbox is checked. In this way, it becomes easier to set colors for labels. In a situation where we need to extend these colors to the markers, this can be done by adding a character prefix of "@" to the colors defined. A sample of this is shown below:

markers HTML 
<div class="checkbox-wrapper">
  <div checkbox data-color="@red|lime" data-label="Unchecked|Checked">

    <div marker="">
      <i style="background-color: currentColor;"></i>
    </div>
    <div marker="">
      <i style="background-color: currentColor;"></i>
    </div>
    
  </div>
  <input type="checkbox">
  <div data-access="label"></div>
</div>

The sample above best reveals how to set the background color of markers by inheriting from the current color assigned to the markers. Similarly, we can inherit color for texts and supported icons. For example, in the structure below, the bootstrap icons will inherit the color from their respective markers.

markers (icons) HTML 
<div  class="checkbox-wrapper" >
  <div checkbox data-color="@red|lime" data-label="Smile|Frown" data-size="20px">

    <div marker="">
      <i class="bi bi-emoji-frown"></i>
    </div>
    <div marker="">
      <i class="bi bi-emoji-smile-fill"></i>
    </div>
    
  </div>
  <input type="checkbox">
  <div data-access="label"></div>
</div>
