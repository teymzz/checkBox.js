Toggle Event


Toggle event is a callback event that is expected to run when the checkbox is toggled. This can be achieved either by global configuration or local configuration using predefined html attribute. The sample below shows how we can define a toggle event globally :

HTML JS (Global) 
<div checkbox="">
  <div marker="" style="background-color:white"></div> 
  <div marker="" style="background-color:lime"></div>
</div>
<input type="checkbox"> <br> 
<div class="response"></div>

let checkbox = new CheckBox(true);

checkbox.check({

  toggle: function(checkbox){
   
     let responseField =  document.querySelector('.response');
     
     responseField.innerHTML = 'checked : ' + checkbox.checked;
   
  }
  
});

  

    

      
 
        

        

      

       
    

    

      

From the sample above, the toggle function will run when the custom checkbox is toggled. Alternatively, we can set up a local custom callback function using the data-func attribute. A sample format is shown below:

HTML (Local) JS 
<div checkbox="" data-func="ontoggle">
  <div marker="" style="background-color:white"></div> 
  <div marker="" style="background-color:lime"></div>
</div>
<input type="checkbox"> <br> 
<div class="response"></div>

new CheckBox(true);

function ontoggle(checkbox){
 
   let responseField =  document.querySelector('.response');
   
   responseField.innerHTML = 'checked : ' + checkbox.checked;
 
}

From the sample html structure above, the data-func attribute was used to specify a local callback function that will be called when the checkbox is toggled. This will also overide any global toggle function predefined.
