CheckBox.js


Disable CheckBox


The checkbox plugin supports disabling of checkboxes by supplying the disabled attribute on the native checkbox. This will prevent any toggle activity on the custom checkbox.

Disable Checkbox HTML 
<div data-role="checkbox-list" data-bind="base">

  <div data-role="checkbox">
    <div checker data-color="@red|lime">
    
      <div marker></div>
     
      <div marker></div>
      
    </div>
    <input disabled type="checkbox">
  </div> 
  
</div>
checkbox (Base Effect)
In the sample above, the custom checkbox cannot be toggled on because its relative native checkbox has been disabled.
