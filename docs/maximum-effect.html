CheckBox.js


Maximum effect


The checkbox maximum effect is a condition that is used to restrict the number of checkboxes that can be selected in a group list. This effect uses the data-max attribute to specify the maximum number of items that can be selected while the data-maxim is a callback function that is executed when the maximum condition is overuled. Consider the sample struture below:

Maximum Effect HTML CSS JS 
  <!--parent checkbox (group)-->
  <div data-role="checkbox-list" data-max="2" data-maxim="overrule">
  
    <!--first checkbox section-->
    <div data-role="checkbox">
      <div checkbox data-anime="bounceIn|bounceIn" data-assign=":marker2" data-color="@#615c988d|#2a7729b8">
      
        <div marker></div>
        <div marker></div>
        
      </div>
      <input type="checkbox" value="C1">
    </div> 
    
    <!--second checkbox section-->
    <div class="rotate" data-role="checkbox">
      <div checkbox data-anime="bounceIn|bounceIn" data-assign=":markers" data-color="@#615c988d|#2b9822b5">
      
        <div marker></div>
        <div marker></div>
        
      </div>
      <input type="checkbox" value="C2">
    </div>
    
    <!--third checkbox section-->
    <div data-role="checkbox">
      <div checkbox data-anime="bounceIn|bounceIn" data-assign=":markers" data-color="@#615c988d|#2cb91cc7">
      
        <div marker></div>
        <div marker></div>
        
      </div>
      <input type="checkbox" value="C3">
    </div>
    
    <!--fourth checkbox section-->
    <div data-role="checkbox">
      <div checkbox data-anime="bounceIn|bounceIn" data-assign=":markers" data-color="@#615c988d|#2cb91cc7">
      
        <div marker></div>
        <div marker></div>
        
      </div>
      <input type="checkbox" value="C4">
    </div>
    
    <!--fifth checkbox section-->
    <div data-role="checkbox">
      <div checkbox data-anime="bounceIn|flash" data-assign=":markers" data-color="@#615c988d|#2cb91cc7">
      
        <div marker></div>
        <div marker></div>
        
      </div>
      <input type="checkbox" value="C5">
    </div>
    
  </div>
  
let checkbox = new CheckBox(true);

checkbox.check({

  assign: {
    alter: (dataAnime) => {
    
        dataAnime.unshift('animated');
        return dataAnime.map( el => 'animate__' + el );
        
    }
  }

});

[data-role="checkbox-list"] {
  display: inline-flex;
  gap: .5em;  
}
  
[data-role="checkbox-list"] [marker] {
  width: 20px;
  height: 20px;  
}
  
[checkbox] {
  display: inline-flex;
  border: dotted 2px;  
  border-radius: 100vh;  
  padding: 4px;  
}
  
[checkbox] [marker] {
  width: 20px;
  height: 20px; 
  border-radius: inherit;    
  background-color: currentColor;    
}

.rotate[checkbox][checked="checked"] {
    animation: rotate 5s infinite linear;;
}

@keyframes rotate {

    0% {
      transform: rotate(0deg);
    }
    
    100% {
      transform: rotate(360deg);
    }
    
}
From the sample above, we have a list of custom checkboxes that are restricted to only a maximum of two selections at a time through the use of data-bind attribute. When more than two selections are made, the overule callback function defined in data-maxim attribute will be triggered.

checkbox.check({

  init: (e) => {

    let rootBox =  document.querySelector(".rateBox");
    
    rootBox.innerHTML = (e.checkedBoxes) + ' boxes checked';

  },
  
  toggle: (e) => e.init()
