Maximum effect


The checkbox maximum effect is a condition that is used to restrict the number of checkboxes that can be selected in a group list. This effect uses the data-max attribute to specify the maximum number of items that can be selected while the data-maxim is a callback function that is executed when the maximum condition is overuled. Consider the sample structure below:

  target: '[checkers]',
  
  init: (e) => {

    let rootBox =  document.querySelector(".rateBox");
    
    rootBox.innerHTML = (e.checkedBoxes) + ' boxes checked';

  },
  
  toggle: (e) => e.init()

});


//** define validate callback function

function validate(e){

    if ( e.checked.length == 3){

        let unchecked = e.unchecked;

        unchecked.forEach( uncheck => {
            uncheck.style.border = "dotted 2px red" 
            setTimeout(() => uncheck.style.border = "dotted 2px #7975a4dc", 500 )
        })

    } else {

        let unchecked = e.unchecked ;
        
        unchecked.forEach( uncheck => {

            uncheck.style.display = "block" 

        }) 
    }

}

From the html structure of the sample above, we have a list of custom checkboxes that are restricted to only a maximum of three selections at a time through the use of data-bind attribute. When more than two selections are made, the validate callback function defined in data-maxim attribute will be triggered.
