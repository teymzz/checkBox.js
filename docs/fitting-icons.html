Fitting icons


Previously, we looked at how to fit images to markers using the data-fit attribute. This attribute may not in most cases will not work well with icons. In order to fit icon sizes, we need to set the font size of the icon to be lesser than the marker's size. This is done through the use of data-fsize attribute. A sample of this is shown below:

HTML CSS JS 
<div checkbox="" data-fit="true" data-size="30px" data-fsize="20px" data-color="@orange|#5271fe">
  <div marker="">
    <div class="fill-marker">
      <i class="bi-watch"></i> 
    </div> 
  </div> 
  
  <div marker="">
    <div class="fill-marker" >
      <i class="bi-watch"></i> 
    </div> 
  </div> 
</div>
<input type="checkbox"> <br> 

.fill-marker {
    width: 100%;
    height: 100%;
    display: grid;
    place-items: center;
}

.fill-marker > * {
    display: grid;
    place-items: center;
    color: currentColor;
}

<script>
new CheckBox();
</script>
SCREEN

From the sample above, notice that the value of the data-size is greater than data-fsize attribute. This is to keep the icons easily within the markers.
