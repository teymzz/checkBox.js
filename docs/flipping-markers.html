Flipping Markers


Double markers can be easily flipped by setting the flip global configuration option or the data-flip local configuration attribute on the custom checkbox. This feature makes one marker hidden at a time while the other is visible. The value of this option is either true or false. The true value means that only one marker will be displayed at a time. For example, the first marker will be displayed when the checkbox is toggled off while the second marker will be displayed when the checkbox is toggled on. However, the false value means that both markers will be visible at the same time. The sample below best reveals how to configure the checkbox globally for flipping markers.

HTML JS(Global) CSS 
<div checkbox="">
  <div marker=""></div> 
  <div marker=""></div>
</div>
<input type="checkbox"> 

<script>

new CheckBox({
    flip: true // toggle double markers
});

</script>    

[checkbox] {
    width: fit-content;
    border-radius: 5px;
    border: solid;
    padding: 4px;
}

[checkbox] > [marker] {
    width: 20px;
    height: 20px;
}

[checkbox][checked="unchecked"] > [marker]:nth-child(1)  {
    background-color: white;
}

[checkbox][checked="checked"] > [marker]:nth-child(2) {
    background-color: lime;
}

Alternatively, we can setup a local configuration from the html structure by using the data-flip attribute as shown below :

HTML (Local) JS CSS 
<div checkbox="" data-flip="true">
  <div marker=""></div> 
  <div marker=""></div>
</div>
<input type="checkbox"> 

<script>

new CheckBox();

</script>    

[checkbox] {
    width: fit-content;
    border-radius: 5px;
    border: solid;
    padding: 4px;
}

[checkbox] > [marker] {
    width: 20px;
    height: 20px;
}

[checkbox][checked="unchecked"] > [marker]:nth-child(1)  {
    background-color: white;
}

[checkbox][checked="checked"] > [marker]:nth-child(2) {
    background-color: lime;
}

This flipping integrated feature makes it easier to create a fast toggle effect for double markers without having to write extra javascript code.
