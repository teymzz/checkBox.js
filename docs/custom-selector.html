CheckBox.js


Custom Selector


By default, the custom checkbox is identified through the “checkbox” attribute. We can easily change this selector by declaring a custom one. An example of this is shown below:

HTML JS#1 JS#2 
<div checkers=""></div>

<input type="checkbox">

new CheckBox({

  target: '[checkers]' //custom box selector

});

let checkbox = new CheckBox(false);

checkbox.check({

  target: '[checkers]' //custom box selector

});

From the javascript code sample above, the [checkers] selector was used to select the target html checkbox which comes immediately before the native checkbox. This will overide the default [checkbox] selector and is also useful in cases where we may require different targets to prevent conflicts if necessary. Any of the two javascript methods above may be applied.
