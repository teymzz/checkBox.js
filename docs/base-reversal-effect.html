Base Reversal Effect


The base "reverse" effect is one in which a checkbox or list of checkboxes are made to negate their parent checkbox. This effect is achieved by setting the data-bind attribute of the native checkbox as "reverse". An example is shown below:

Base Reversal Effect HTML 
<--parent checkbox (grand)-->
<div data-role="checkbox-list" data-bind="base" style="flex-direction: column;">
                  
  <div class="">
      <--first checkbox section-->
      <div data-role="checkbox" class="in-flex wrap">
        <div checker data-anime="fadeIn" data-assign=":marker2" data-label="Select all|Unselect all" data-color="@orange|lime" class="in-flex">
        
          <div marker class="b-currentColor"></div>
          
          <div marker class="b-currentColor"></div>
          
        </div>
        <input type="checkbox" value="radio1">
      </div>
      <div class="">
           (parent)
      </div>
  </div>

  <div class="">
      <--second checkbox section-->
      <div data-role="checkbox" class="in-flex wrap">
        <div checker data-anime="fadeIn" data-assign=":marker2" data-color="@red|lime" class="in-flex">
        
          <div marker class="b-currentColor"></div>
          
          <div marker class="b-currentColor"></div>
          
        </div>
        <input type="checkbox" data-bind="reverse" value="radio2">
      </div>
      <div class="">
           data-bind = 'reverse'
      </div> 
  </div>

  <div class="">
      <--third checkbox section-->
      <div data-role="checkbox" class="in-flex wrap">
        <div checker data-anime="fadeIn" data-assign=":marker2" data-color="@red|lime" class="in-flex">
        
          <div marker class="b-currentColor"></div>
          
          <div marker class="b-currentColor"></div>
          
        </div>
        <input type="checkbox" data-bind="reverse" value="radio3">
      </div> 
      <div class="">
           data-bind = 'reverse'
      </div>
  </div>

  <div class="">
      <--third checkbox section-->
      <div data-role="checkbox" class="in-flex wrap">
        <div checker data-anime="fadeIn" data-assign=":marker2" data-color="@red|lime" class="in-flex">
        
          <div marker class="b-currentColor"></div>
          
          <div marker class="b-currentColor"></div>
          
        </div>
        <input type="checkbox" value="radio4">
      </div> 
      <div class="">
          (no data-bind)
      </div>
  </div>

  <div class="">
      <--third checkbox section-->
      <div data-role="checkbox" class="in-flex wrap">
        <div checker data-anime="fadeIn" data-assign=":marker2" data-color="@red|lime" class="in-flex">
        
          <div marker class="b-currentColor"></div>
          
          <div marker class="b-currentColor"></div>
          
        </div>
        <input type="checkbox" data-bind="free" value="radio5">
      </div> 
      <div class="">
           data-bind = 'free'
      </div>
  </div>

</div>
Base Reversal Effect
  (parent)
  data-bind = 'reverse'
  data-bind = 'reverse'
  (no data-bind)
  data-bind = 'free'
In the sample above, the first checkbox is the parent checkbox. The relationship between the second and third checkbox is that they are always the opposite of the parent checkbox which is not the case for the fourth checkbox which is a direct reflection of the parent. The fifth checkbox is however completely detached from this relationship. The exhibition of the base reversal effect is seen in both the second and third checkboxes.
