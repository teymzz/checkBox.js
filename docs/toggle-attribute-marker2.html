CheckBox.js


Toggle Attributes - :marker2


This option specifies that the value from an attribute on the custom checkbox should be assigned into another attribute on the second marker when the checkbox is checked. A sample of this is shown below

HTML JS (GLOBAL) 
<div checkers class="foo" data-switch="bar">
  <div marker style="background-color: red;"></div>
  <div marker style="background-color: green;"></div>
</div>
<input type="checkbox">

<script>
new CheckBox({
  assign: {
      target: '[checkers]',
      attr: ['data-switch', 'class', ':marker2']
  }    
});
</script>

From the sample html structure, the "bar" value will be added into the class attribute of the second marker when the checkbox is checked and removed when unchecked due to the plugin's global configuration. Alternatively, we can do this locally using the data-assign attribute.

HTML (LOCAL) CSS JS 
<div checkers class="foo" data-switch="bar" data-assign=":marker2|data-switch|class">
  <div marker style="background-color: red;"></div>
  <div marker style="background-color: green;"></div>
</div>
<input type="checkbox">

[checkers] {
  width: fit-content;
  border-radius: 100vh;    
  border: dotted 2.5px #514e68; 
  padding: 4px;    
}

[checkers] > [marker] {
  width: 60px;
  height: 2px;    
  background-color: transparent;
  transition: width .5s ease-in-out, 
              background-color .5s ease-in-out;
}

[checkers] > [marker] {
  border-radius: 100vh;
  background-color: yellow;
  margin: 4px;
}

[checkers] > [marker]:first-child {
  width: 30px;
  background-color: red;
}

[checkers] > .bar[marker] {
  width: 120px;                         
  background-color: lime;
}

<script>
new CheckBox();
</script>
SCREEN

The sample marker above will increase the length of the second marker to 120px when the checkbox is checked due to the value "bar" that is added into the second marker's class attribute.
