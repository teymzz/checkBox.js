Radio Effect


The radio effect is a custom checkbox effect that is similar to the input radio buttons and it is specified by adding an extra attribute of data-bind with a value of "radio" checklist element identified with the data-role="checkbox-list" attribute. An example is shown below.

Radio Effect HTML CSS JS 
<!--parent checkbox (group)-->
<div data-role="checkbox-list" data-bind="radio">

  <!--first checkbox section-->
  <div data-role="checkbox">
    <div checkbox data-anime="fadeIn" data-assign=":marker2" data-color="@#615c988d|lime">
    
      <div marker></div>
      <div marker></div>
      
    </div>
    <input type="checkbox" value="C1">
  </div> 
  
  <!--second checkbox section-->
  <div class="rotate" data-role="checkbox">
    <div checkbox data-anime="rubberBand infinite" data-assign=":marker2" data-color="@#615c988d|gold">
    
      <div marker></div>
      <div marker></div>
      
    </div>
    <input type="checkbox" value="C2">
  </div>
  
  <!--third checkbox section-->
  <div data-role="checkbox">
    <div checkbox data-anime="bounceIn|flash infinite" data-assign=":markers" data-color="@#615c988d|lime">
    
      <div marker></div>
      <div marker></div>
      
    </div>
    <input type="checkbox" value="C3">
  </div>
  
</div>

[data-role="checkbox-list"] {
  display: inline-flex;
  gap: .5em;  
}
  
[data-role="checkbox-list"] [marker] {
  width: 20px;
  height: 20px;  
}
  
[checkbox] {
  display: inline-flex;
  border: dotted 2px;  
  border-radius: 100vh;  
  padding: 4px;  
}
  
[checkbox] [marker] {
  width: 20px;
  height: 20px; 
  border-radius: inherit;    
  background-color: currentColor;    
}

.rotate[checkbox][checked="checked"] {
    animation: rotate 5s infinite linear;;
}

@keyframes rotate {

    0% {
      transform: rotate(0deg);
    }
    
    100% {
      transform: rotate(360deg);
    }
    
}

let checkbox = new CheckBox(false);

checkbox.check({

    assign: {
        alter: (dataAnime) => {
        
            /* use a default attribute value */
            dataAnime.unshift('animated');
            return dataAnime.map( el => 'animate__' + el );
            
        }
    }

});

SCREEN

From the sample above, we can see how easy it is to customize and animate checkboxes using the checkbox plugin. In order to use the animate.css animation classes, from the javascript section, we smartly modified the destination values such that each animation class is assigned prefix of "animated__". This reduces the length of values we needed to supply on the html structure and also improves code readability.
