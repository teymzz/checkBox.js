Custom Marker


By default, the custom checkbox marker is selected through the [marker] attribute selector. We can easily overide this selector by declaring a new custom selector. An example of this is shown below

HTML CSS JS 
<div checkbox>
  <div btn=""></div> 
  <div btn=""></div> 
</div>
<input type="checkbox">

[checkbox] {
    width: fit-content;
    border-radius: 5px;
    border: solid;
    padding: 4px;
}

[checkbox] > [btn] {
    width: 20px;
    height: 20px;
}

[checkbox][checked="checked"] > [btn]:nth-child(1) {
    opacity: .1;
}

[checkbox]:not([checked="checked"]) > [btn]:nth-child(2) {
    opacity: .1;
}

[checkbox] > [btn]:nth-child(1)  {
    background-color: white;
}

[checkbox] > [btn]:nth-child(2) {
    background-color: lime;
}

let checkbox = new CheckBox(false);

checkbox.check({

  marker: '[btn]' //custom marker selector

});
SCREEN

From the sample above, in the javascript section, the [btn] selector is used to target the markers. This will overide the default [marker] selector.
