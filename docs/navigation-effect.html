CheckBox.js


Navigation Effect


The checkbox navigation effects are otherwise known as button effects because they are only binded to navigation controller buttons and defined within the data-nav attribute. There are three basic effects that are supported by the data-nav attribute and these are the hover, click and press effects. The sample structure below is the format used to define a navigation effect.

HTML 
<div id="checklist-1" data-role="checkbox-list" data-nav="prev|next|stop|effect">

  <!-- add some control buttons --> 

</div>

In the sample above, we are required to define the navigation controller ids before finally defining the effect we want to apply. While the first two arguments "prev" and "next" may be required, we don't necessarily need to define the stop id before defining the effect. This can be done easily by using the format below:

HTML 
<div id="checklist-1" data-role="checkbox-list" data-nav="prev|next||effect">

  <!-- add some control buttons --> 

</div>

Now that we understand the structure accepted for defining our effects, let discuss more on each of these effects.

  • click This effect is the default effect for navigation buttons. It is simply used to trigger the action of a previous or next specified navigation button when the button is clicked upon. We can easily animate this by specifying the animation time. For example, a click-500 will add an interval of 500ms animation effect when the button is clicked upon.
  • hover This effect uses the mouseenter event to trigger the action of a previous or next button when the button is hovered upon. We can also animate this by specifying the animation time. For example, an hover-500 will add an interval of 500ms animation effect when the button is hovered upon. This interval is automatically terminated when the button is no longer being hovered upon.
  • press This effect triggers the action of a previous or next button when the button is pressed down. This effect at most times require an animation interval. For example, a press-500 will add an interval of 500ms animation effect only when the button is pressed down.

The example below is one in which the press-200 effect is used.

HTML JS 
  
  <div id="checklist-1" data-role="checkbox-list" data-bind="radio" data-nav="prev|next|stop|press-200">
  
    <!--first checkbox section-->
    <div data-role="checkbox">
      <div checkbox data-color="@#615c988d|lime">
        <div marker></div>
        <div marker></div>
      </div>
      <input type="checkbox">
    </div> 
    
    <!--second checkbox section-->
    <div data-role="checkbox">
      <div checkbox data-color="@#615c988d|lime">
        <div marker></div>
        <div marker></div>
      </div>
      <input type="checkbox">
    </div>
    
    <!--third checkbox section-->
    <div data-role="checkbox">
      <div checkbox data-color="@#615c988d|lime">
        <div marker></div>
        <div marker></div>
      </div>
      <input type="checkbox">
    </div>
  
  </div>  
  
  <!-- add some control buttons -->  
  <div class="controls">
    <button id="prev">&#10094;</button>
    <button id="stop">&#9724;</button>
    <button id="next">&#10095;</button>
  </div>  
  

  let checkbox = new CheckBox(false);
  
  checkbox = checkbox.check({ flip: true });
