Rating effect


The rating toggle effect includes a list of custom checkboxes that exists together purposely for rating items. In this effect, when a checkbox is checked, all relative custom checkboxes to its left side are automatically toggled on while those to the right are toggled off. This is achieved by setting the value of data-bind attribute to "rating". A sample of this is displayed below:

Rating Effect HTML CSS JS 
  <!--parent checkbox (group)-->
  <div data-role="checkbox-list" data-bind="rating">
  
    <!--first checkbox section-->
    <div data-role="checkbox">
      <div checkbox data-anime="bounceIn|bounceIn" data-assign=":marker2" data-color="@#615c988d|#2a7729b8">
      
        <div marker></div>
        <div marker></div>
        
      </div>
      <input type="checkbox" value="C1">
    </div> 
    
    <!--second checkbox section-->
    <div class="rotate" data-role="checkbox">
      <div checkbox data-anime="bounceIn|bounceIn" data-assign=":markers" data-color="@#615c988d|#2b9822b5">
      
        <div marker></div>
        <div marker></div>
        
      </div>
      <input type="checkbox" value="C2">
    </div>
    
    <!--third checkbox section-->
    <div data-role="checkbox">
      <div checkbox data-anime="bounceIn|bounceIn" data-assign=":markers" data-color="@#615c988d|#2cb91cc7">
      
        <div marker></div>
        <div marker></div>
        
      </div>
      <input type="checkbox" value="C3">
    </div>
    
    <!--fourth checkbox section-->
    <div data-role="checkbox">
      <div checkbox data-anime="bounceIn|bounceIn" data-assign=":markers" data-color="@#615c988d|#2cb91cc7">
      
        <div marker></div>
        <div marker></div>
        
      </div>
      <input type="checkbox" value="C4">
    </div>
    
    <!--fifth checkbox section-->
    <div data-role="checkbox">
      <div checkbox data-anime="bounceIn|flash" data-assign=":markers" data-color="@#615c988d|#2cb91cc7">
      
        <div marker></div>
        <div marker></div>
        
      </div>
      <input type="checkbox" value="C5">
    </div>
    
  </div>
  
let checkbox = new CheckBox(false);

checkbox.check({

  assign: {
    alter: (dataAnime) => {
    
        dataAnime.unshift('animated');
        return dataAnime.map( el => 'animate__' + el );
        
    }
  }

});

[data-role="checkbox-list"] {
  display: inline-flex;
  gap: .5em;  
}
  
[data-role="checkbox-list"] [marker] {
  width: 20px;
  height: 20px;  
}
  
[checkbox] {
  display: inline-flex;
  border: dotted 2px;  
  border-radius: 100vh;  
  padding: 4px;  
}
  
[checkbox] [marker] {
  width: 20px;
  height: 20px; 
  border-radius: inherit;    
  background-color: currentColor;    
}

.rotate[checkbox][checked="checked"] {
    animation: rotate 5s infinite linear;;
}

@keyframes rotate {

    0% {
      transform: rotate(0deg);
    }
    
    100% {
      transform: rotate(360deg);
    }
    
}

let checkbox = new CheckBox(false);

checkbox.check({

  init: (e) => {

    let rootBox =  document.querySelector(".rateBox");
    
    rootBox.innerHTML = (e.checkedBoxes) + ' boxes checked';

  },
  
  toggle: (e) => e.init()

});

We can enable touch swipe or mouse swipe event for rating checkboxes which makes it easier to swipe forward or backward on the custom checkboxes. This can be done by setting the attribute data-swipe to "enabled" on the checkbox list group. This effect is applied in the sample below

  <!--parent checkbox (group)-->
